# taskflow-system
## Mudança de Escopo

Durante o desenvolvimento do TaskFlow System, identificou-se a necessidade crítica de classificar as tarefas por níveis de urgência. 

Por conta disso, o escopo foi refinado para incluir o atributo **prioridade** (Alta, Média, Baixa) no gerenciamento das tarefas, permitindo uma melhor tomada de decisão da equipe.
